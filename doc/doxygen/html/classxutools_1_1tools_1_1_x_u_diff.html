  cf_match,

  report

Node-visitor action that processes the node of type 'cf_match' to generate an xudiff report.

Here we apply the production associated with the reference in this cf_match node and compare the two parse trees using Zhang and Shasha's algorithm.

Parameters:
self
[in]cf_matchThe 'cf_match' parse-tree node to visit
[in]indentThe current indentation level in the parse tree.
[in]reportThe current xudiff report. This should really be an in,out param
Returns:
The xudiff report If the 'cf_match' node is None or empty, then we've reached a base case on recursion. Just return the report
